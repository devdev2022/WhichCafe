# 카페어디

## 개요
사용자의 현재 위치를 중심으로 24시간 운영하는 카페의 위치를 보여주는 서비스 입니다. 

### 주요기능
- 사용자의 위치를 중심으로 직경 2km 이내에 있는 카페의 위치를 확인할 수 있습니다. 
- 주소 검색을 통해 자신이 갈 예정인 장소에 있는 카페정보를 확인할 수 있습니다. 
- 즐겨찾기를 통해 카페정보를 저장할 수 있습니다.
